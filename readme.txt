This is a plugin it is a stream of  the latest technological news in realtime.

== Description ==
just install this and see the tech news coming to your site (inside the widget) in real-time updating every second..

get the latest tech news in real time in a very cool way ... it has more than 1000 headlines a hour from all the major news sources.
just take a look at the **screenshots** its a great widget for every website.

**if you think your site is a great tech news source and you dont want to install the plugin please send us an email team(at)zvoid(.)com with
your website link and we will take a look...**

== Installation ==

1. Upload `realtimetechnews.php` to the `/wp-content/plugins/` directory or to `/wp-content/plugins/realtime-tech-news/`
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Place widget `realtime technological news` to Sidebar and configure it.
